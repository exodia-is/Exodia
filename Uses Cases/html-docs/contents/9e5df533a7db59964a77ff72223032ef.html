Escenarios: CU-01: Gestionar Clientes

Gestionar Clientes (Flujo básico)

  1. El caso de uso inicia cuando el administrador seleciona la opción de gestionar clientes

  2. El administrador puede o no seleccionar 'Reistrar cliente'

  3. El admisnistrador puede o no seleccionar 'Actualizar cliente'

  4. El administrador puede o no seleccionar 'Consultar cliente'

  5. El caso de uso finaliza

Gestionar Clientes (Flujos alternos)

2.1 El administrador selecionó la opción de 'Registrar cliente'

2.1.1 El sistema ejecuta el caso de uso CU-02: Registrar Cliente'

2.1.2 El caso de uso finaliza

3.1 El administrador seleccionó la opción de 'Actualizar Cliente'

3.1.1 El sistema ejecuta el caso de uso CU-03: Actualizar Cliente

3.1.2 El caso de uso finaliza

4.1 El administrador seleccionó la opción de 'Consular Cliente'

4.1.1 El sistema ejecuta el caso de uso CU-04: Consultar Cliente

4.1.2 El caso de uso finaliza
