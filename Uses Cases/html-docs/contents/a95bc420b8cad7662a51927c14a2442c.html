Escenarios: CU-04: Consultar Clientes

Consultar Cliente (Flujo básico)

1- El caso de uso se inicia cuando Admin selecciona Actualizar cliente

2- El administrador puede o no seleccionar 'Consulta General'

3- El administrador puede o no seleccionar 'Consulta especifica'

4- El caso de uso finaliza

Flijos alternos

2.1 Si el administrador selecciona la opcion 'Consulta genearl'

2.1.1 se ejecuta el caso de uso CU-05: Consulta general

3.1- Si el administrador seleccionó 'Consulta especifica'

3.1.1 se ejecuta el caso de uso CU-05: Consulta especifica
