Description

Escenarios: CU-06: Consultar Especifica

Consulta especifica (Flujo básico)

1- El caso de uso se inicia cuando Admin selecciona 'Consulta especifica'

2- El sistema despliega un formulario de consulta con los siguientes datos: * Cod_id (Documento)

3- El administrador llena el formulario

4- El administrador da click en continuar

5- El sistema muestra una vista del cliente

Flijos alternos

5.1 Si sistema verifica que los campos esten completos

5.1.1 el sistema muestra 'Faltan campos por llenar'

5.2- El sistema verifica que el usuario este registrado

5.2.1 El sistema muestra el mensaje 'El usuario no está registrado'

5.2.2 Em sistema muestra las opciones: Salir Registrar

     5.2.2.1 El adinistrador seleccionó Salir

       5.2.2.1.1 El sistema regresa a la pagina de origen

     5.2.2.2  El administrador seleccionó 'Registrar'

         5.2.2.2.1 Se ejecuta el caso de uso CU-02: Registrar  Cliente
